# Medical AIO Research (Korea)

한국 의료기관의 생성형 AI 검색 노출에 대한 종단 측정 및 분석 프로젝트.

## Researcher
- **이름**: [김민정], RN
- **소속**: [독립 연구자]
- **전문 영역**: 임상 간호 + 의료 마케팅 + AI 측정 시스템 개발
- **연구 시작일**: 2026-05-08

## 연구 배경
ChatGPT, Claude, Perplexity, Gemini, HyperCLOVA X 등 생성형 AI 검색 환경에서 
한국 의료기관이 어떻게 인용·추천되는가에 대한 정량적 종단 데이터는 현재 부재하다. 
본 프로젝트는 자체 개발 측정 도구를 통해 이 공백을 채우는 것을 목적으로 한다.

## 연구 자산

### 측정 인프라 (FORMEDI GEO Tracker)
- Python 백엔드 (22 modules) + 웹 인터페이스 (16 pages)
- SQLite 기반 데이터 저장 (17 tables)
- 6개 AI 엔진 어댑터: ChatGPT, Claude, Gemini, Perplexity, Grok, AIO
- 자동 스케줄 측정 (일 1회) + 즉시 분석 모드
- 의료광고심의 컴플라이언스 메타데이터 (개발 중)

### 측정 현황
- **베이스라인 측정 시작**: 2026-05-XX (실측 API 연동 완료 후 갱신 예정)
- **이전 시스템 검증 단계 데이터**: 2,380건 (시뮬레이션 응답, 인프라 안정성 검증용. 분석 대상에서 제외)

## 방법론

### 핵심 지표
- **Citation Frequency**: 의료기관이 AI 응답에 언급되는 빈도
- **Mention Rank**: 응답 내 언급 순위
- **Engine Coverage**: 측정 대상 엔진 중 언급되는 비율
- **(개발 중) Compliance-Adjusted Score**: 의료법 56조 컴플라이언스를 가중치로 반영한 종합 점수

### 측정 프로토콜
- 표준 환자 페르소나 기반 자연어 쿼리셋
- 일 1회 자동 측정 (동일 시각, 동일 프롬프트 셋)
- 응답 텍스트 + 출처 URL + 메타데이터 전수 저장

## 윤리 및 컴플라이언스

본 연구는 의료법 제56조(의료광고 규제), 개인정보보호법, 의료법 제27조(무면허 의료행위 금지) 
조항을 우선 고려하여 설계된다. 측정·분석 과정에서 수집되는 모든 데이터는 
환자 개인정보를 포함하지 않으며, 의료기관 정보는 익명화 후 집계된 형태로만 공개된다.
